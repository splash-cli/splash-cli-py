# Splash Cli PY
> Splash Cli tool in Python

## How to
Download the [`main.py`](main.py) file, and run it via 

```sh
  $ python 3 main.py
```

#### ⚠️ Warns
- It works only on macOS systems.
- It downloads only random photos.
- No help menu
- No flags
- No Arguments
